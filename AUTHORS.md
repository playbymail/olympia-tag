# Authors & Credits

## Original game

Olympia was created by **Rich Skrenta** (Copyright 1993 — the original sources
are public domain). The engine in this repository is **TAG** ("The Age of
Gods"), the largest and newest of the four Olympia play-by-mail strategy game
engines — it extends the base game with artifacts, religion, quests, hero
skills, effects, mining, ranger skills, and ship crafting, and descends from G3.
